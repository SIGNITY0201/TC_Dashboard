## 대시보드 통합 버전

### ver1.0 (2026-04-07)
- 기존 3개 별도 GitHub repo를 **1개 repo로 통합** (대시보드+견적서+계약서)
- 멀티 페이지 구조: index.html(대시보드) → quotation/(견적서) → contract/(계약서)
- 고객 탭 / 관리자 탭 분리
- 계약서에 **조이블라인드** 시공파트너 추가
- 견적서에 **Glass Folding** 품목 추가 (350,000원/짝, 시공비 연동)
- **PWA 지원** (manifest.json, service-worker.js → 앱 설치 가능)
- **배포:** GitHub Pages (SIGNITY0201/TC_Dashboard)

### ver2.0 (2026-04-29) ← 현재 운영
- **데크 견적 로직 전면 교체:**
  - 색상 카테고리 시스템 도입 (자연석 st / 인조석 st / 우드 st, 총 8색)
  - 색상 기반 단가: 자연석 85,000원 / 인조석·우드 95,000원 (per m²)
  - 규격: 상판 가로×세로 + 첼판 가로×높이 → m² 자동계산 (기존 600×600 고정에서 변경)
  - 옵션 칸에 색상 샘플 이미지 표시 (8종 png)
- **스톤 스텝 (+각관)** 품목 신규 추가
- 시공비 변경: 페데스탈 35,000원/m², 각관 75,000원/m² (데크·스텝 동일)
- 고객 전화번호 **자동 하이픈** 포맷 (010-0000-0000)
- 고객 탭 메뉴 정리 (견적 도구 제거 → 관리자 전용)
- file:// 프로토콜 호환: 모든 링크에 index.html 명시
- service-worker.js 캐시 v2.0 업데이트 (데크 이미지 포함)
- 데크 전용 견적서(deck-quotation/) 생성 후 통합 견적서에 합치고 삭제
